This project is an example of a movie recommender system using user-based and/or item-based collaborative filtering methods.

## Features

- Generates personalized movie recommendations based on user-movie rating data
- Includes fundamental data analysis and recommendation algorithms
- Developed in Python

## Requirements

- Python 3.x
- NumPy
- pandas
- For additional dependencies, please check the code files in the repository

## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/eray-yuztyurk/collaborative-filtering-movie-recommender.git
    ```
2. Navigate to the project directory:
    ```bash
    cd collaborative-filtering-movie-recommender
    ```
3. Install the required Python packages:
    ```bash
    pip install -r requirements.txt
    ```
    Alternatively, you can manually install the required libraries.

## Usage

To run the project, execute the main Python file:
```bash
python main.py
```
Input and output formats are described in the relevant code files.
